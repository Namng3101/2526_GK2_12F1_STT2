# NHOM NHOM STORE – WEBSITE BÁN BÁNH NGỌT

## 1. Chủ đề

Website **Nhom Nhom Store** là một trang web giới thiệu và bán các loại bánh ngọt như cookie, cupcake, cheesecake, donut, mochi và tiramisu.
Người dùng có thể:

* Xem danh sách các loại bánh
* Tùy chỉnh một số loại bánh theo sở thích
* Thêm sản phẩm vào giỏ hàng
* Tra cứu tình trạng đơn hàng

Website được xây dựng bằng **HTML, CSS và JavaScript cơ bản** nhằm tạo ra một giao diện thân thiện, dễ sử dụng và mang phong cách ngọt ngào phù hợp với chủ đề bánh ngọt.
